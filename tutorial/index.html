Explore the App

This is a reactive web application that consumes the Twitter Streaming API and allows users to do a real-time search within all tweets that contain a set of terms.

Setup

You need a running instance of ElasticSearch. On a mac it is the easiest to use homebrew, on other operating systems please see the installation instructions on the ElasticSearch website.

brew install elasticsearch

You then run

elasticsearch -f

Then, before running the play application for the first time, you need to create the percolation index in ElasticSearch:

: curl -XPUT localhost:9200/queries

Twitter API consumer key and access token are required to consume the Twitter Streaming API. You need to create a Twitter application and store keys and secrets in a twitter.conf file, using the commented out section in the application.conf as a template.
